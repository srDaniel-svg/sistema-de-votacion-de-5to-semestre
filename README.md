# Sistema de Votación - 5to Semestre UAGRM

Este es un portal de votación interactivo desarrollado para la Facultad de Ingeniería en Ciencias de la Computación de la Universidad Autónoma Gabriel René Moreno (UAGRM).

## Características

- **Ranking Interactivo**: Visualización de candidatos con efectos dinámicos y diseño optimizado para móviles.
- **Sistema de Votación**: Integración con Supabase para registro de votos en tiempo real.
- **Panel de Administración**: Gestión completa de candidatos, estadísticas, imágenes y videos.
- **Diseño Responsivo**: Experiencia fluida tanto en escritorio como en dispositivos móviles.
- **Autenticación**: Sistema de login seguro para votantes y administradores.

## Tecnologías Utilizadas

- **Frontend**: React + TypeScript + Vite
- **Estilos**: Tailwind CSS + Shadcn/UI
- **Backend/Base de Datos**: Supabase
- **Iconos**: Lucide React
